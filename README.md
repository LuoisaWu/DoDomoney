# Dodomoney

Dodomoney（钱多多）是一款 AI 智能记账软件。第一版目标不是做一个只有页面的演示，而是搭建可扩展的软件骨架：桌面客户端负责交互，后端服务负责账单、预算、AI 解析、数据分析与持久化。

## 项目定位

- 自然语言记账：用户输入“午饭 22 元”，系统自动识别金额、类别、时间与备注。
- AI 财务助手：默认助手“账小喵”，支持不同语气和角色风格。
- 账单管理：支持收支记录的增删改查、分类纠错和后续学习。
- 预算分析：支持月度统计、分类占比、预算提醒与异常消费分析。
- 图片扩展：后续接入小票 OCR、支付截图识别、食物识别与热量估算。

## 技术架构

```text
Dodomoney
├─ apps
│  ├─ api       FastAPI 后端服务，负责业务逻辑、数据库、AI 编排
│  └─ desktop   Electron + React 桌面客户端
├─ docs         架构、路线图、小组协作说明
├─ shared       前后端共享契约、类型与分类字典
└─ scripts      开发辅助脚本
```

## 快速开始

### 1. 启动后端

```powershell
cd apps/api
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
python -m app.main
```

后端默认运行在 `http://127.0.0.1:8000`。

### 2. 启动桌面端

```powershell
cd apps/desktop
npm install
npm run dev
```

桌面端开发模式会启动 Vite 和 Electron。

## 小组分工建议

- 后端组：FastAPI 接口、SQLite 数据模型、预算和统计服务。
- AI 组：自然语言解析、分类规则、提示词、后续模型接入。
- 桌面端组：聊天式记账界面、账单列表、统计页、设置页。
- 产品与测试组：需求拆解、交互文案、测试用例、演示数据。
