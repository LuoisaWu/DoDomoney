# 开发路线图

## Milestone 1：自然语言记账闭环

- 完成聊天输入框。
- 完成 `/chat/record` 接口。
- 规则版 AI 解析金额、类型、分类、描述。
- 写入 SQLite 并返回账单卡片数据。
- 支持账单列表查看。

## Milestone 2：账单管理与分类纠错

- 支持账单编辑、删除、筛选。
- 建立分类字典和子分类。
- 用户纠正分类后记录偏好。
- 下一次类似描述优先使用用户偏好。

## Milestone 3：预算与统计

- 月度收入、支出、结余统计。
- 分类消费占比。
- 月预算与分类预算。
- 超预算提醒。

## Milestone 4：AI 角色系统

- 支持助手名称、头像、语气、回复长度。
- 提供可爱陪伴型、毒舌吐槽型、温柔鼓励型、理性顾问型。
- 后端根据角色配置生成回复。

## Milestone 5：图片与语音能力

- 图片上传接口。
- 小票 OCR 与支付截图识别。
- 食物照片识别与热量估算。
- 语音转文字记账。

## Milestone 6：产品化

- 打包 Windows / macOS / Linux 桌面应用。
- 增加数据备份和导入导出。
- 增加自动更新。
- 增加隐私说明和本地数据保护策略。
